4. CSS 속성의 상속(Inheritance) 및 크기 단위(px / % / em / rem / vw / vh) 실습

개발자 도구(F12 -> Elements -> Computed)를 열어 상속 관계와 각 박스의 실제 연산 크기를 검증해 보세요.

[실습 1] CSS 속성 상속 검증

부모 상자 (.parent-box)

파란 테두리와 안쪽 여백을 부여했습니다. 이 텍스트는 부모의 글자색(파랑)을 자동 상속 받았습니다.

자식 상자 (.child-box)

하지만 부모의 초록색 테두리(Border)나 안쪽 여백(Padding)은 자식에게 복사·상속되지 않습니다.

[실습 2] 폰트 크기 단위 대조 (em vs rem)

부모 폰트 크기를 20px로 선언한 콘텐트 영역

이곳은 1.5em입니다. (부모 20px * 1.5 = 30px)

이곳은 2rem입니다. (Root 16px * 2 = 32px)

[실습 3] 레이아웃 및 뷰포트 크기 단위 대조 (px / % / vw / vh)

1) px (절대 고정 단위)

150px x 50px

2) % (부모 영역 대비 백분율 단위)

부모 영역 (width: 400px;)

자식 영역 (width: 50% = 200px)

3) vw / vh (브라우저 화면 뷰포트 기준 단위)

브라우저 창 크기를 가로/세로로 조절하면 아래 초록색 박스의 크기가 실시간으로 반응하여 늘어나거나 줄어듭니다.

가로 50vw x 세로 10vh
